Please note that the subdirectories of `src` (such as `src/latest`) are Lean projects that (last I checked) build as a whole.  However, be aware that it might take many hours to finish.

This repository collects Lean proofs for various mathematical statements from multiple sources:

* [Erdős problems](ErdosProblems/README.md)
* [MathOverflow](MathOverflow/README.md)
* [100 Theorems](HundredTheorems/README.md)
