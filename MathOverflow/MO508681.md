**Definition.**
The game is defined as follows.

Alice receives an initial number $A_0\sim \mathrm{Unif}[0,1]$ and Bob receives an initial number $B_0\sim \mathrm{Unif}[0,1]$, independently. Alice moves first.

On a turn of a player whose current number is $x$, that player chooses one of the following two actions.

*Showdown.* The game ends immediately. The player with the larger current number wins. Alice's payoff is $+1$ if her current number is larger and $-1$ if Bob's current number is larger.

*Continue.* The player draws a new random number $U\sim \mathrm{Unif}[0,1]$, independent of all previous random variables, replaces the current number $x$ by $\max\{x,U\}$, and the turn passes to the other player.

If showdown never occurs, then the payoff is $0$ to both players.

Since all random variables are continuous, the event of a tie at showdown has probability $0$, so its assigned payoff is irrelevant.

**Definition.**
A strategy for Alice is a behavioral strategy
```math
\alpha=(\alpha_n)_{n\ge 1},
```
where $\alpha_n:[0,1]^n\to[0,1]$ is measurable, and $\alpha_n(x_1,\dots,x_n)$ is the probability that Alice chooses showdown on her $n$th turn when her own sampled numbers so far are $x_1,\dots,x_n$. A strategy for Bob is defined similarly.

For strategies $\alpha,\beta$, let $u(\alpha,\beta)$ denote Alice's expected payoff.

**Definition.**
A pair of strategies $(\alpha^\ast,\beta^\ast)$ is an *equilibrium pair* if
```math
u(\alpha,\beta^\ast)\le u(\alpha^\ast,\beta^\ast)\le u(\alpha^\ast,\beta)
```
for all strategies $\alpha,\beta$. In that case, the common quantity $u(\alpha^\ast,\beta^\ast)$ is called the *value* of the game.

**Definition.**
For $r\in[0,1]$, let $\sigma_r$ denote the stationary threshold strategy defined by the rule: on each turn, if the player's current number is $x$, then choose showdown when $x\ge r$ and choose continue when $x<r$.

**Lemma 1.**
Fix $r\in(0,1)$, and suppose that a player uses $\sigma_r$. For $k\ge 0$, let $X_k$ be the player's current number at the beginning of that player's $(k+1)$st turn, conditioned on the event that the player chose continue on each of the first $k$ turns. Then
```math
\mathbb P(X_k\le x)=\begin{cases}\dfrac{x^{k+1}}{r^k}, & 0\le x<r,\\[1.2ex]x, & r\le x\le 1.\end{cases}
```
In particular,
```math
\mathbb P(X_k\lt r)=r,
```
and, conditional on $\{X_k\ge r\}$, the random variable $X_k$ is uniformly distributed on $[r,1]$.

*Proof*.
For $k=0$, the current number is initially uniform on $[0,1]$, so the formula holds.

Assume the formula holds for some $k\ge 0$. On the event that the player continued on each of the first $k+1$ turns, one has $X_k<r$, and the new current number is
```math
X_{k+1}=\max(X_k,U),
```
where $U\sim \mathrm{Unif}[0,1]$ is independent of $X_k$.

If $0\le x<r$, then
```math
\mathbb P(X_{k+1}\le x)=\mathbb P(X_k\le x,\ U\le x\mid X_k<r)
=
\frac{\mathbb P(X_k\le x)\,x}{\mathbb P(X_k<r)}
=
\frac{x\cdot \dfrac{x^{k+1}}{r^k}}{r}
=
\frac{x^{k+2}}{r^{k+1}}.
```
If $r\le x\le 1$, then on the event $X_k<r$ the condition $X_{k+1}\le x$ is equivalent to $U\le x$, so
```math
\mathbb P(X_{k+1}\le x)=x.
```
This proves the stated formula.

Now
```math
\mathbb P(X_k<r)=\mathbb P(X_k\le r)=r.
```
For $r\le x\le 1$,
```math
\mathbb P(X_k\le x\mid X_k\ge r)
=
\frac{\mathbb P(r\le X_k\le x)}{\mathbb P(X_k\ge r)}
=
\frac{x-r}{1-r},
```
so $X_k\mid (X_k\ge r)$ is uniform on $[r,1]$.
□

**Lemma 2.**
Fix $r\in(0,1)$, and suppose Bob uses $\sigma_r$. For $k\ge 0$ and $x\in[0,1]$, let $V_k(x)$ be the supremum of Alice's expected payoff in the subgame that starts from the state in which it is Alice's turn, Alice's current number is $x$, and each player has already had $k$ turns. Then $V_k(x)\in[-1,1]$ is well defined, and
```math
V_k(x)=\max\{D_k(x),\, C_rV_{k+1}(x)\},
```
where
```math
D_k(x)=
\begin{cases}
2\dfrac{x^{k+1}}{r^k}-1, & 0\le x<r,\\[1.2ex]
2x-1, & r\le x\le 1,
\end{cases}
```
and, for a bounded function $f:[0,1]\to\mathbb R$,
```math
(C_rf)(x)
=
x\bigl((1-r)g(x)+rf(x)\bigr)
+
\int_x^1 \bigl((1-r)g(u)+rf(u)\bigr)\,du,
```
with
```math
g(x)=
\begin{cases}
-1, & 0\le x<r,\\[1ex]
\dfrac{2x-1-r}{1-r}, & r\le x\le 1.
\end{cases}
```

*Proof*.
By Lemma 1, on each Bob turn, conditional on the game having reached that turn, Bob chooses continue with probability $r$. Hence the probability that Bob continues on each of his first $N$ turns is $r^N$, which tends to $0$ as $N\to\infty$. Therefore Bob stops almost surely in finite time, regardless of Alice's strategy. In particular, the game ends almost surely, so $V_k(x)$ is well defined and belongs to $[-1,1]$.

If Alice chooses showdown immediately, then her payoff is
```math
\mathbb P(\text{Alice wins})-\mathbb P(\text{Alice loses})
=
2\mathbb P(\text{Bob's current number}<x)-1.
```
Since Bob's current number after $k$ previous Bob turns has the distribution given by Lemma 1, this equals $D_k(x)$.

If Alice chooses continue, let
```math
Z=\max(x,U),
```
where $U\sim \mathrm{Unif}[0,1]$ is independent of all previous randomness. Then
```math
\mathbb P(Z=x)=x,
```
and $Z$ has density $1$ on $[x,1]$.

On Bob's next turn, Bob stops with probability $1-r$. Conditional on that event, Bob's stopping number is uniformly distributed on $[r,1]$ by Lemma 1. Therefore Alice's expected payoff if Bob stops on that turn is $g(Z)$, where
```math
g(Z)=
\begin{cases}
-1, & Z<r,\\[1ex]
\dfrac{2Z-1-r}{1-r}, & Z\ge r.
\end{cases}
```
If Bob does not stop, which happens with probability $r$, then after Bob's move the game returns to a state of the same form, namely Alice's turn, Alice's current number equal to $Z$, and each player having already had $k+1$ turns. By definition, the optimal continuation value from that state is $V_{k+1}(Z)$.

Thus, if Alice chooses continue at $x$, her optimal expected payoff is
```math
\mathbb E\bigl[(1-r)g(Z)+rV_{k+1}(Z)\bigr],
```
which is exactly $C_rV_{k+1}(x)$ when written in terms of the distribution of $Z$.

Alice may choose either showdown or continue at the present turn. Hence
```math
V_k(x)=\max\{D_k(x),\, C_rV_{k+1}(x)\}.
```

**Lemma 3.**
Fix $r\in(0,1)$. The recursion
```math
W_k(x)=\max\{D_k(x),\, C_rW_{k+1}(x)\}
\qquad (k\ge 0,\ 0\le x\le 1)
```
has at most one uniformly bounded solution, where uniformly bounded means
```math
\sup_{k\ge 0}\sup_{0\le x\le 1}|W_k(x)|<\infty.
```

*Proof*.
Suppose $(W_k)$ and $(\widetilde W_k)$ are uniformly bounded solutions. Set
```math
\delta_k=\sup_{0\le x\le 1}|W_k(x)-\widetilde W_k(x)|.
```
Since the map $y\mapsto \max\{a,y\}$ is $1$-Lipschitz,
```math
\delta_k
\le
\sup_{0\le x\le 1}\bigl|C_rW_{k+1}(x)-C_r\widetilde W_{k+1}(x)\bigr|.
```
For bounded functions $f,h$ and every $x\in[0,1]$,
```math
\bigl|C_rf(x)-C_rh(x)\bigr|
=
r\left|x(f(x)-h(x))+\int_x^1 (f(u)-h(u))\,du\right|
\le r\|f-h\|_\infty.
```
Therefore
```math
\delta_k\le r\,\delta_{k+1}.
```

Let
```math
M=\sup_{j\ge 0}\delta_j<\infty.
```
Iterating the inequality $\delta_k\le r\,\delta_{k+1}$ gives
```math
\delta_k\le r^m\delta_{k+m}\le Mr^m
\qquad (m\ge 0).
```
Letting $m\to\infty$, since $0<r<1$, one obtains $\delta_k=0$. Thus $W_k=\widetilde W_k$ for every $k$.
□

**Lemma 4.**
Let
```math
r=\sqrt2-1,
\qquad
c=2r-1.
```
Define
```math
w(x)=
\begin{cases}
c, & 0\le x<r,\\[1ex]
2x-1, & r\le x\le 1.
\end{cases}
```
Then for every $k\ge 0$ and every $x\in[0,1]$,
```math
w(x)=\max\{D_k(x),\, C_rw(x)\}.
```

*Proof*.
Since $r=\sqrt2-1$, one has
```math
r^2+2r-1=0,
\qquad
c=2r-1=-r^2.
```

First assume $0\le x<r$. Then
```math
D_k(x)=2\frac{x^{k+1}}{r^k}-1\le 2x-1<2r-1=c=w(x).
```
Also,
```math
(1-r)g(u)+rw(u)=
\begin{cases}
rc+r-1, & 0\le u<r,\\[1ex]
2(1+r)u-(1+2r), & r\le u\le 1.
\end{cases}
```
Hence
\begin{align*}
C_rw(x)
&=
x(rc+r-1)+\int_x^r (rc+r-1)\,du
 +\int_r^1 \bigl(2(1+r)u-(1+2r)\bigr)\,du \\
&=
r(rc+r-1)+r^2(1-r).
\end{align*}
Substituting $c=2r-1$ gives
```math
C_rw(x)=r(2r^2-1)+r^2(1-r)=r^3+r^2-r.
```
Using $r^2+2r-1=0$, one has
```math
r^3+r^2-r=r(r^2+r-1)=-r^2=c.
```
Thus
```math
C_rw(x)=c=w(x),
```
and therefore
```math
w(x)=\max\{D_k(x),\, C_rw(x)\}.
```

Now assume $r\le x\le 1$. Then
```math
D_k(x)=2x-1=w(x).
```
For $u\ge r$,
```math
(1-r)g(u)+rw(u)=2(1+r)u-(1+2r),
```
so
\begin{align*}
C_rw(x)
&=
x\bigl(2(1+r)x-(1+2r)\bigr)
+\int_x^1 \bigl(2(1+r)u-(1+2r)\bigr)\,du \\
&=
(1+r)x^2-r.
\end{align*}
Therefore
```math
w(x)-C_rw(x)=2x-1-(1+r)x^2+r.
```
Since
```math
(1+r)^2=2,
\qquad
r(1+r)=1-r,
```
the right-hand side factors as
```math
2x-1-(1+r)x^2+r=(1+r)(1-x)(x-r)\ge 0.
```
Hence
```math
w(x)=D_k(x)\ge C_rw(x),
```
and so
```math
w(x)=\max\{D_k(x),\, C_rw(x)\}.
```
This proves the lemma.
□

**Theorem.**
Let
```math
r=\sqrt2-1.
```
Then $(\sigma_r,\sigma_r)$ is a pair of equilibrium strategies to this game. Equivalently, in every round each player should choose showdown exactly when the current number is at least $\sqrt2-1$; at the boundary $x=\sqrt2-1$, either action is optimal.

The value of the game is
```math
r^2=(\sqrt2-1)^2=3-2\sqrt2
```
to Alice.

*Proof*.
Fix Bob's strategy to be $\sigma_r$. By Lemma 2, Alice's optimal value functions $(V_k)$ satisfy
```math
V_k(x)=\max\{D_k(x),\, C_rV_{k+1}(x)\}.
```
Every $V_k$ is bounded by $1$ in absolute value. By Lemma 3, this recursion has at most one uniformly bounded solution. By Lemma 4, the sequence $V_k(x)=w(x)$ is such a solution. Therefore
```math
V_k(x)=w(x)
\qquad\text{for all }k\ge 0,\ 0\le x\le 1.
```

Hence, against Bob's strategy $\sigma_r$, Alice's optimal action is: continue when $x<r$, choose showdown when $x>r$, and either action when $x=r$. This is exactly the strategy $\sigma_r$, up to the irrelevant tie-breaking at $x=r$.

By symmetry, the same statement holds with the roles of Alice and Bob interchanged. Therefore $(\sigma_r,\sigma_r)$ is a pair of equilibrium strategies to this game.

It remains to compute the payoff. At the initial state it is Alice's turn, her initial number is uniformly distributed on $[0,1]$, and no player has yet moved. Thus
```math
u(\sigma_r,\sigma_r)=\int_0^1 w(x)\,dx.
```
Since $w(x)=c$ on $[0,r)$ and $w(x)=2x-1$ on $[r,1]$,
```math
u(\sigma_r,\sigma_r)
=
\int_0^r c\,dx+\int_r^1 (2x-1)\,dx
=
r(2r-1)+(r-r^2)=r^2.
```
Therefore the value of the game is
```math
r^2=(\sqrt2-1)^2=3-2\sqrt2
```
to Alice.
□
