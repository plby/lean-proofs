# One-sided limsup sets for middle-a Cantor sets

## 1. The middle-a Cantor set and the one-sided limsup sets

**Definition 1.1.**

Let $a \in (0,1)$. Define $C_{a,0} = [0,1]$. Having defined $C_{a,n}$ as a finite union of closed intervals, obtain $C_{a,n+1}$ by removing from each interval in $C_{a,n}$ its open middle subinterval of relative length $a$. The middle $a$ Cantor set is
```math
C_a := \bigcap_{n=0}^\infty C_{a,n}.
```

Now fix $\alpha \in (0,1)$. The one-sided limsup sets are
```math
C_\alpha^+ := \limsup_{a \to \alpha_+} C_a
:= \bigcap_{\varepsilon \gt 0}\ \bigcup_{\alpha \lt a \lt \alpha+\varepsilon} C_a,
```
and
```math
C_\alpha^- := \limsup_{a \to \alpha_-} C_a
:= \bigcap_{\varepsilon \gt 0}\ \bigcup_{\alpha-\varepsilon \lt a \lt \alpha} C_a.
```

It is convenient to reparametrise by
```math
r = \frac{1-a}{2} \in \left(0,\frac12\right),
\qquad
a = 1-2r.
```
For fixed $\alpha \in (0,1)$ we write
```math
\rho := \frac{1-\alpha}{2}.
```
Since $a \mapsto r = (1-a)/2$ is decreasing, the limits $a \to \alpha_+$ and $a \to \alpha_-$ correspond to $r \to \rho_-$ and $r \to \rho_+$ respectively.

Write
```math
\Sigma := \{0,1\}^{\mathbb{N}},
\qquad
\Sigma_n := \{0,1\}^n \quad (n \ge 1),
\qquad
\Sigma_* := \bigcup_{n \ge 1} \Sigma_n.
```
If $u \in \Sigma_n$, we write $|u| = n$. As usual, $0^n$ and $1^n$ denote the all-zero and all-one words of length $n$.

For $\omega = (\omega_1,\omega_2,\dots) \in \Sigma$ and $n \in \mathbb{N}$, we write $\omega|_n := \omega_1\cdots\omega_n \in \Sigma_n$. For a finite word $u$, the symbols $u0^\infty$ and $u1^\infty$ mean the infinite sequences obtained by appending infinitely many zeros or ones to $u$.

For $r \in (0,1/2)$ define the two contractions
```math
f_{0,r}(x) := rx,
\qquad
f_{1,r}(x) := 1-r+rx
\qquad
(x \in \mathbb{R}).
```
For a word $u = u_1\cdots u_n \in \Sigma_n$, define
```math
f_{u,r} := f_{u_1,r}\circ\cdots\circ f_{u_n,r},
\qquad
I_u(r) := f_{u,r}([0,1]).
```

**Proposition 1.2.**

Let $r \in (0,1/2)$, and put $a = 1-2r$.

**(1)** For every $u = u_1\cdots u_n \in \Sigma_n$,
```math
f_{u,r}(x) = (1-r)\sum_{k=1}^n u_k r^{k-1} + r^n x
\qquad
(x \in \mathbb{R}).
```
Hence
```math
I_u(r)
=
\left[
(1-r)\sum_{k=1}^n u_k r^{k-1},
\ (1-r)\sum_{k=1}^n u_k r^{k-1}+r^n
\right],
```
so $|I_u(r)| = r^n$.

**(2)** The $n\text{th}$ stage of the middle $a$ construction is
```math
C_{a,n} = \bigcup_{u \in \Sigma_n} I_u(r).
```
Consequently,
```math
C_a = \bigcap_{n=1}^\infty\ \bigcup_{u \in \Sigma_n} I_u(r).
```

**(3)** Define
```math
\pi_r(\omega) := (1-r)\sum_{n=1}^\infty \omega_n r^{n-1}
\qquad
(\omega \in \Sigma).
```
Then
```math
C_a = \pi_r(\Sigma).
```
Moreover, for every $u \in \Sigma_n$,
```math
L_u(r) := \pi_r(u0^\infty),
\qquad
R_u(r) := \pi_r(u1^\infty)
```
are the left and right endpoints of $I_u(r)$, so that
```math
I_u(r) = [L_u(r),R_u(r)].
```

**(4)** The map $\pi_r : \Sigma \to C_a$ is a homeomorphism (where $\Sigma$ has the product topology). In particular, each point of $C_a$ has a unique code.

**Proof.**

Part (1) is an immediate induction on $n$.

For part (2), the first stage of the middle $a$ construction is
```math
[0,r]\cup[1-r,1] = I_0(r)\cup I_1(r).
```
Applying the same operation inside each surviving interval shows inductively that the $n\text{th}$ stage is exactly the union of the $2^n$ intervals $I_u(r)$, $u \in \Sigma_n$.

For part (3), let $\omega \in \Sigma$. The intervals $I_{\omega|_n}(r)$ are nested and have diameters $r^n \to 0$, hence their intersection consists of a single point. By part (1), that point is
```math
\lim_{n\to\infty} f_{\omega|_n,r}(0)
=
\lim_{n\to\infty}(1-r)\sum_{k=1}^n \omega_k r^{k-1}
=
(1-r)\sum_{k=1}^\infty \omega_k r^{k-1}
=
\pi_r(\omega).
```
Thus every $\pi_r(\omega)$ belongs to $C_a$. Conversely, if $x \in C_a$, then for each $n$ there is a unique word $u_n \in \Sigma_n$ such that $x \in I_{u_n}(r)$; these words are compatible, so they define a unique $\omega \in \Sigma$ with $u_n = \omega|_n$ for all $n$, and then $x = \pi_r(\omega)$. This proves $C_a = \pi_r(\Sigma)$. The endpoint formulas follow by taking $\omega = u0^\infty$ and $\omega = u1^\infty$.

For part (4), continuity of $\pi_r$ in $\omega$ follows from the uniformly convergent series. To prove injectivity, let $\omega \ne \eta$ and let $m$ be the first index at which they differ. Then $\omega|_m \ne \eta|_m$, so the level-$m$ intervals $I_{\omega|_m}(r)$ and $I_{\eta|_m}(r)$ are distinct members of the construction and therefore disjoint. Since $\pi_r(\omega) \in I_{\omega|_m}(r)$ and $\pi_r(\eta) \in I_{\eta|_m}(r)$, we must have $\pi_r(\omega) \ne \pi_r(\eta)$. Thus $\pi_r$ is a continuous bijection from the compact space $\Sigma$ onto the Hausdorff space $C_a$, hence a homeomorphism.

∎

For later use, note that
```math
C_\alpha^+ = \limsup_{r\to\rho_-} C_{1-2r},
\qquad
C_\alpha^- = \limsup_{r\to\rho_+} C_{1-2r}.
```

**Lemma 1.3.**

Fix $r_0 \in (0,1/2)$. Then the map
```math
[0,r_0]\times \Sigma \ni (r,\omega) \longmapsto \pi_r(\omega)
```
is continuous.

**Proof.**

For $N \in \mathbb{N}$, define
```math
S_N(r,\omega) := (1-r)\sum_{n=1}^N \omega_n r^{n-1}.
```
Each $S_N$ is continuous. Moreover, for $(r,\omega) \in [0,r_0]\times\Sigma$,
```math
\bigl|\pi_r(\omega)-S_N(r,\omega)\bigr|
\le
(1-r)\sum_{n=N+1}^\infty r^{n-1}
\le
r_0^N.
```
Thus $S_N$ converges uniformly on $[0,r_0]\times\Sigma$ to $(r,\omega)\mapsto \pi_r(\omega)$, so $(r,\omega)\mapsto \pi_r(\omega)$ is continuous.

∎

**Lemma 1.4.**

Let $\rho \in (0,1/2)$ and let $r_n \to \rho$. If $x \in C_{1-2r_n}$ for every $n$, then $x \in C_{1-2\rho}$.

In particular,
```math
\limsup_{r\to \rho_-} C_{1-2r}\subseteq C_{1-2\rho},
\qquad
\limsup_{r\to \rho_+} C_{1-2r}\subseteq C_{1-2\rho}.
```

**Proof.**

For each $n$, choose $\omega^{(n)} \in \Sigma$ such that
```math
x = \pi_{r_n}(\omega^{(n)}).
```
By a diagonal subsequence argument there are a subsequence $(n_k)$ and some $\omega \in \Sigma$ such that $\omega^{(n_k)} \to \omega$ in $\Sigma$. Lemma 1.3 gives
```math
x
=
\lim_{k\to\infty}\pi_{r_{n_k}}(\omega^{(n_k)})
=
\pi_\rho(\omega)
\in
C_{1-2\rho}.
```

∎

**Lemma 1.5.**

Let $\omega \in \Sigma$.

**(1)** If $\omega_1 = 0$ and $\omega \ne 0^\infty$, then $r \mapsto \pi_r(\omega)$ is strictly increasing on $[0,1/2)$.

**(2)** If $\omega_1 = 1$ and $\omega \ne 1^\infty$, then $r \mapsto \pi_r(\omega)$ is strictly decreasing on $[0,1/2)$.

**Proof.**

Assume first that $\omega_1 = 0$ and $\omega \ne 0^\infty$. Write the set of indices at which $\omega_n = 1$ as a disjoint union of maximal blocks
```math
\{n \in \mathbb{N} : \omega_n = 1\}
=
\bigcup_j [a_j,b_j),
```
where each $a_j \ge 2$ and each $b_j \in \mathbb{N}\cup\{\infty\}$. Then
```math
\pi_r(\omega)
=
(1-r)\sum_j\sum_{n=a_j}^{b_j-1} r^{n-1}
=
\sum_{j: b_j \lt \infty} \bigl(r^{a_j-1}-r^{b_j-1}\bigr)
+
\sum_{j: b_j=\infty} r^{a_j-1}.
```
Thus it is enough to check that every nonzero summand is strictly increasing on $[0,1/2)$.

The terms $r^{a_j-1}$ are obviously strictly increasing. Now fix integers $2 \le a \lt b$, and set
```math
g_{a,b}(r) := r^{a-1}-r^{b-1}.
```
For $0 \lt r \lt 1/2$,
```math
g_{a,b}'(r)
=
r^{a-2}\bigl((a-1)-(b-1)r^{\,b-a}\bigr).
```
Write $m = b-a \ge 1$. Since $r \lt 1/2$,
```math
(b-1)r^m \lt (b-1)2^{-m}.
```
Also
```math
2^m(a-1)-(b-1)
=
2^m(a-1)-(a+m-1)
=
(2^m-1)(a-1)-m
\ge
0,
```
because $a-1 \ge 1$ and $2^m-1 \ge m$. Hence $(b-1)2^{-m} \le a-1$, and therefore $g'_{a,b}(r) \gt 0$ for $0 \lt r \lt 1/2$. So every nonzero summand is strictly increasing, and therefore $\pi_r(\omega)$ is strictly increasing.

For part (2), observe that
```math
\pi_r(1-\omega)=1-\pi_r(\omega),
```
where $1-\omega := (1-\omega_1,1-\omega_2,\dots)$. If $\omega_1 = 1$ and $\omega \ne 1^\infty$, then $1-\omega$ starts with $0$ and is not $0^\infty$, so part (1) shows that $r \mapsto \pi_r(1-\omega)$ is strictly increasing. Hence $r \mapsto \pi_r(\omega)$ is strictly decreasing.

∎

**Theorem 1.6.**

Let $\alpha \in (0,1)$ and $\rho = (1-\alpha)/2$. Define
```math
\begin{aligned}
E_\rho^+
:={}&
\{R_u(\rho): u \in \Sigma_*,\ u_1=0\} \\
&\cup
\{L_u(\rho): u \in \Sigma_*,\ u_1=1\}.
\end{aligned}
```
Equivalently,
```math
\begin{aligned}
E_\rho^+
:={}&
\{\pi_\rho(u1^\infty): u \in \Sigma_*,\ u_1=0\} \\
&\cup
\{\pi_\rho(u0^\infty): u \in \Sigma_*,\ u_1=1\}.
\end{aligned}
```
Then
```math
C_\alpha^+ = C_\alpha \setminus E_\rho^+.
```

**Proof.**

Since $C_\alpha^+ = \limsup_{r\to\rho_-} C_{1-2r}$, Lemma 1.4 yields
```math
C_\alpha^+ \subseteq C_{1-2\rho} = C_\alpha.
```

We first show that $E_\rho^+ \cap C_\alpha^+ = \varnothing$.

Suppose $x = R_u(\rho)$ for some $u \in \Sigma_n$ with $u_1 = 0$. The interval $I_u(\rho)$ is one of the $2^n$ level-$n$ intervals in the construction of $C_\alpha$, and $x$ is its right endpoint. Since distinct level-$n$ intervals are disjoint, we have
```math
d := \operatorname{dist}\!\left(
x,\bigcup_{\substack{v\in\Sigma_n\\ v\ne u}} I_v(\rho)
\right)
\gt 0.
```
For each $v \in \Sigma_n$, the endpoints $L_v(r)$ and $R_v(r)$ are continuous in $r$, so for some $\delta \gt 0$ we have
```math
\operatorname{dist}\bigl(x,I_v(r)\bigr)\ge \frac d2
\qquad
\text{for all } v \ne u
\quad\text{and all } r \in (\rho-\delta,\rho+\delta).
```
On the other hand, $R_u(r) = \pi_r(u1^\infty)$ is strictly increasing by Lemma 1.5, so
```math
R_u(r) \lt R_u(\rho) = x
\qquad
(r \in (\rho-\delta,\rho)).
```
Hence for such $r$ we have $x \notin I_u(r)$, and also $x \notin I_v(r)$ for all $v \ne u$. Therefore
```math
x \notin \bigcup_{v\in\Sigma_n} I_v(r) \supseteq C_{1-2r}
\qquad
(r \in (\rho-\delta,\rho)).
```
So $x \notin C_\alpha^+$.

The case $x = L_u(\rho)$ with $u_1 = 1$ is analogous: now $L_u(r) = \pi_r(u0^\infty)$ is strictly decreasing, so $L_u(r) \gt x$ for $r \lt \rho$ sufficiently close to $\rho$, and the same continuity argument for the other level-$n$ intervals shows that $x \notin C_{1-2r}$ for all such $r$. Thus indeed
```math
E_\rho^+ \cap C_\alpha^+ = \varnothing.
```

It remains to prove
```math
C_\alpha \setminus E_\rho^+ \subseteq C_\alpha^+.
```
Take $x \in C_\alpha \setminus E_\rho^+$, and let $\omega \in \Sigma$ be its unique code, so that $x = \pi_\rho(\omega)$.

If $x = 0$ or $x = 1$, then $x \in C_a$ for every $a \in (0,1)$, hence $x \in C_\alpha^+$. We may therefore assume $x \notin \{0,1\}$.

Assume first that $\omega_1 = 0$. Since $x \notin E_\rho^+$, the code $\omega$ is not eventually $1$. Fix $\varepsilon \gt 0$, and choose
```math
s \in (\max\{0,\rho-\varepsilon\},\rho).
```
By Lemma 1.5,
```math
\pi_s(\omega) \lt \pi_\rho(\omega) = x.
```
Let $u_n := \omega|_n$. Then $\pi_s(\omega) \in I_{u_n}(s) = [L_{u_n}(s),R_{u_n}(s)]$ and $|I_{u_n}(s)| = s^n$, so
```math
R_{u_n}(s)\longrightarrow \pi_s(\omega).
```
Hence for some $n$,
```math
R_{u_n}(s) \lt x.
```
On the other hand, because $\omega$ is not eventually $1$, we have
```math
x \lt R_{u_n}(\rho) = \pi_\rho(u_n1^\infty)
\qquad
\text{for every } n.
```
The function $r \mapsto R_{u_n}(r)$ is continuous, so there exists
```math
r \in (s,\rho)\subset(\rho-\varepsilon,\rho)
```
such that
```math
R_{u_n}(r) = x.
```
Thus
```math
x = R_{u_n}(r) = \pi_r(u_n1^\infty) \in C_{1-2r}.
```
Since $\varepsilon \gt 0$ was arbitrary, $x \in C_\alpha^+$.

Now assume that $\omega_1 = 1$. Since $x \notin E_\rho^+$, the code $\omega$ is not eventually $0$. Fix $\varepsilon \gt 0$ and choose again
```math
s \in (\max\{0,\rho-\varepsilon\},\rho).
```
By Lemma 1.5,
```math
\pi_s(\omega) \gt \pi_\rho(\omega) = x.
```
Let $u_n := \omega|_n$. Then $\pi_s(\omega) \in I_{u_n}(s) = [L_{u_n}(s),R_{u_n}(s)]$ and $|I_{u_n}(s)| = s^n$, so
```math
L_{u_n}(s)\longrightarrow \pi_s(\omega).
```
Hence for some $n$,
```math
L_{u_n}(s) \gt x.
```
Because $\omega$ is not eventually $0$, we also have
```math
L_{u_n}(\rho) = \pi_\rho(u_n0^\infty) \lt x
\qquad
\text{for every } n.
```
By continuity of $r \mapsto L_{u_n}(r)$, there exists
```math
r \in (s,\rho)\subset(\rho-\varepsilon,\rho)
```
such that
```math
L_{u_n}(r) = x.
```
Thus
```math
x = L_{u_n}(r) = \pi_r(u_n0^\infty) \in C_{1-2r}.
```
Again $\varepsilon \gt 0$ was arbitrary, so $x \in C_\alpha^+$.

This completes the proof.

∎

**Theorem 1.7.**

Let $\alpha \in (0,1)$ and $\rho = (1-\alpha)/2$. Define
```math
\begin{aligned}
E_\rho^-
:={}&
\{L_u(\rho): u \in \Sigma_*,\ u_1=0,\ u \ne 0^{|u|}\} \\
&\cup
\{R_u(\rho): u \in \Sigma_*,\ u_1=1,\ u \ne 1^{|u|}\}.
\end{aligned}
```
Equivalently,
```math
\begin{aligned}
E_\rho^-
:={}&
\{\pi_\rho(u0^\infty): u \in \Sigma_*,\ u_1=0,\ u \ne 0^{|u|}\} \\
&\cup
\{\pi_\rho(u1^\infty): u \in \Sigma_*,\ u_1=1,\ u \ne 1^{|u|}\}.
\end{aligned}
```
Then
```math
C_\alpha^- = C_\alpha \setminus E_\rho^-.
```

**Proof.**

Since $C_\alpha^- = \limsup_{r\to\rho_+} C_{1-2r}$, Lemma 1.4 gives
```math
C_\alpha^- \subseteq C_{1-2\rho} = C_\alpha.
```

We first show that $E_\rho^- \cap C_\alpha^- = \varnothing$.

Suppose $x = L_u(\rho)$ for some $u \in \Sigma_n$ with $u_1 = 0$ and $u \ne 0^n$. Then $x$ is the left endpoint of the level-$n$ interval $I_u(\rho)$. As in the proof of Theorem 1.6,
```math
d := \operatorname{dist}\!\left(
x,\bigcup_{\substack{v\in\Sigma_n\\ v\ne u}} I_v(\rho)
\right)
\gt 0,
```
and continuity of the finitely many endpoint functions yields $\delta \gt 0$ such that
```math
\operatorname{dist}\bigl(x,I_v(r)\bigr)\ge \frac d2
\qquad
\text{for all } v \ne u
\quad\text{and all } r \in (\rho-\delta,\rho+\delta).
```
Since $u0^\infty$ begins with $0$ and is not $0^\infty$, Lemma 1.5 shows that $L_u(r) = \pi_r(u0^\infty)$ is strictly increasing. Therefore
```math
L_u(r) \gt L_u(\rho) = x
\qquad
(r \in (\rho,\rho+\delta)).
```
So for such $r$ we have $x \notin I_u(r)$, and also $x \notin I_v(r)$ for $v \ne u$. Hence
```math
x \notin \bigcup_{v\in\Sigma_n} I_v(r) \supseteq C_{1-2r}
\qquad
(r \in (\rho,\rho+\delta)).
```
Thus $x \notin C_\alpha^-$.

The case $x = R_u(\rho)$ with $u_1 = 1$ and $u \ne 1^{|u|}$ is analogous. Indeed $u1^\infty$ begins with $1$ and is not $1^\infty$, so Lemma 1.5 shows that $R_u(r) = \pi_r(u1^\infty)$ is strictly decreasing, hence $R_u(r) \lt x$ for $r \gt \rho$ close to $\rho$. The same continuity argument for the other level-$n$ intervals then shows that $x \notin C_{1-2r}$ for all such $r$. Therefore
```math
E_\rho^- \cap C_\alpha^- = \varnothing.
```

It remains to prove
```math
C_\alpha \setminus E_\rho^- \subseteq C_\alpha^-.
```
Take $x \in C_\alpha \setminus E_\rho^-$, and let $\omega \in \Sigma$ be the unique code of $x$, so that $x = \pi_\rho(\omega)$.

Again $0,1 \in C_a$ for every $a$, so we may assume $x \notin \{0,1\}$.

Assume first that $\omega_1 = 0$. Since $x \notin E_\rho^-$, the code $\omega$ is not eventually $0$. Fix $\varepsilon \gt 0$, and choose
```math
s \in \bigl(\rho,\min\{\rho+\varepsilon,1/2\}\bigr).
```
By Lemma 1.5,
```math
\pi_s(\omega) \gt \pi_\rho(\omega) = x.
```
Let $u_n := \omega|_n$. Then $\pi_s(\omega) \in I_{u_n}(s) = [L_{u_n}(s),R_{u_n}(s)]$ and $|I_{u_n}(s)| = s^n$, so
```math
L_{u_n}(s)\longrightarrow \pi_s(\omega).
```
Hence for some $n$,
```math
L_{u_n}(s) \gt x.
```
Because $\omega$ is not eventually $0$, we also have
```math
L_{u_n}(\rho) = \pi_\rho(u_n0^\infty) \lt x
\qquad
\text{for every } n.
```
By continuity of $r \mapsto L_{u_n}(r)$, there exists
```math
r \in (\rho,s)\subset(\rho,\rho+\varepsilon)
```
such that
```math
L_{u_n}(r) = x.
```
Hence
```math
x = L_{u_n}(r) = \pi_r(u_n0^\infty) \in C_{1-2r}.
```
Since $\varepsilon \gt 0$ was arbitrary, $x \in C_\alpha^-$.

Now assume that $\omega_1 = 1$. Since $x \notin E_\rho^-$, the code $\omega$ is not eventually $1$. Fix $\varepsilon \gt 0$ and choose
```math
s \in \bigl(\rho,\min\{\rho+\varepsilon,1/2\}\bigr).
```
By Lemma 1.5,
```math
\pi_s(\omega) \lt \pi_\rho(\omega) = x.
```
Let $u_n := \omega|_n$. Then $\pi_s(\omega) \in I_{u_n}(s) = [L_{u_n}(s),R_{u_n}(s)]$ and $|I_{u_n}(s)| = s^n$, so
```math
R_{u_n}(s)\longrightarrow \pi_s(\omega).
```
Hence for some $n$,
```math
R_{u_n}(s) \lt x.
```
Because $\omega$ is not eventually $1$, we also have
```math
x \lt R_{u_n}(\rho) = \pi_\rho(u_n1^\infty)
\qquad
\text{for every } n.
```
By continuity of $r \mapsto R_{u_n}(r)$, there exists
```math
r \in (\rho,s)\subset(\rho,\rho+\varepsilon)
```
such that
```math
R_{u_n}(r) = x.
```
Hence
```math
x = R_{u_n}(r) = \pi_r(u_n1^\infty) \in C_{1-2r}.
```
Thus $x \in C_\alpha^-$.

The proof is complete.

∎

**Remark 1.8.**

Because the coding is unique, Theorems 1.6 and 1.7 admit the following symbolic reformulations:
```math
C_\alpha^+
=
\pi_\rho\left(
\{\omega \in \Sigma : \omega_1 = 0 \text{ and } \omega \text{ is not eventually } 1\}
\cup
\{\omega \in \Sigma : \omega_1 = 1 \text{ and } \omega \text{ is not eventually } 0\}
\right),
```
and
```math
C_\alpha^-
=
\{0,1\}\cup
\pi_\rho\left(
\{\omega \in \Sigma : \omega_1 = 0 \text{ and } \omega \text{ is not eventually } 0\}
\cup
\{\omega \in \Sigma : \omega_1 = 1 \text{ and } \omega \text{ is not eventually } 1\}
\right).
```

## 2. Hausdorff and box dimensions

We recall the standard definitions. For $E \subset \mathbb{R}$, $s \ge 0$, and $\delta \gt 0$, let
```math
\mathcal{H}_\delta^s(E)
:=
\inf\left\{
\sum_{j=1}^\infty (\operatorname{diam}(U_j))^s :
E \subset \bigcup_{j=1}^\infty U_j,\ \operatorname{diam}(U_j)\le\delta
\right\},
```
and
```math
\mathcal{H}^s(E) := \lim_{\delta\downarrow 0}\mathcal{H}_\delta^s(E).
```
The Hausdorff dimension of $E$ is
```math
\dim_H(E) := \inf\{s \ge 0 : \mathcal{H}^s(E)=0\}.
```

For bounded $E \subset \mathbb{R}$ and $\delta \gt 0$, let
```math
N_\delta(E)
:=
\min\left\{
m \in \mathbb{N} :
E \subset \bigcup_{j=1}^m U_j,\ \operatorname{diam}(U_j)\le\delta
\right\}.
```
The lower and upper box dimensions are
```math
\underline{\dim}_B(E)
:=
\liminf_{\delta\downarrow 0}\frac{\log N_\delta(E)}{-\log\delta},
\qquad
\overline{\dim}_B(E)
:=
\limsup_{\delta\downarrow 0}\frac{\log N_\delta(E)}{-\log\delta}.
```

**Lemma 2.1 (Mass distribution principle).**

Let $E \subset \mathbb{R}$, and let $\mu$ be a Borel probability measure supported on $E$. Suppose there exist $s \ge 0$, $C \gt 0$, and $\delta_0 \gt 0$ such that
```math
\mu(U)\le C(\operatorname{diam}(U))^s
\qquad
\text{whenever } 0 \lt \operatorname{diam}(U)\le\delta_0.
```
Then $\mathcal{H}^s(E)\ge 1/C$. In particular, $\dim_H(E)\ge s$.

**Proof.**

Fix $0 \lt \delta \le \delta_0$, and let $\{U_j\}$ be any countable cover of $E$ with $\operatorname{diam}(U_j)\le\delta$. Then
```math
1 = \mu(E)\le \sum_j \mu(U_j)\le C\sum_j (\operatorname{diam}(U_j))^s.
```
Taking the infimum over all such covers gives
```math
\mathcal{H}_\delta^s(E)\ge \frac1C.
```
Letting $\delta\downarrow 0$ yields $\mathcal{H}^s(E)\ge 1/C$.

∎

**Theorem 2.2.**

Let $a \in (0,1)$, and put
```math
r = \frac{1-a}{2},
\qquad
s = \frac{\log 2}{-\log r}
= \frac{\log 2}{\log(2/(1-a))}.
```
Then
```math
\dim_H(C_a)
=
\underline{\dim}_B(C_a)
=
\overline{\dim}_B(C_a)
=
s.
```

**Proof.**

By Proposition 1.2, the set $C_a$ is the union of $2^n$ level-$n$ intervals $I_u(r)$, each of length $r^n$.

Every gap between adjacent level-$n$ intervals is obtained by applying $f_{v,r}$ to the original middle gap of length $a$, for some word $v$ of length $0,1,\dots,n-1$. Hence every level-$n$ gap has length $ar^m$ for some $m \in \{0,\dots,n-1\}$, and the smallest such gap has length
```math
ar^{n-1}.
```

**Hausdorff upper bound.**

Covering $C_a$ by the $2^n$ level-$n$ intervals gives
```math
\mathcal{H}_{r^n}^s(C_a)\le 2^n(r^n)^s = (2r^s)^n = 1.
```
Hence $\mathcal{H}^s(C_a)\le 1$, so $\dim_H(C_a)\le s$.

**Hausdorff lower bound.**

Let $\nu$ be the $(1/2,1/2)$ Bernoulli measure on $\Sigma$, and let
```math
\mu := (\pi_r)_\#\nu
```
be its pushforward to $C_a$. Then for every word $u \in \Sigma_n$,
```math
\mu(I_u(r))
=
\nu(\{\omega \in \Sigma : \omega|_n = u\})
=
2^{-n}
=
r^{sn}.
```

Now fix an interval $J \subset \mathbb{R}$ with $0 \lt |J| \lt a$. Choose $n \in \mathbb{N}$ such that
```math
ar^n \le |J| \lt ar^{n-1}.
```
Since the smallest gap between level-$n$ intervals has length $ar^{n-1}$, the interval $J$ can meet at most one level-$n$ interval. Therefore
```math
\mu(J)\le 2^{-n}=r^{sn}\le a^{-s}|J|^s.
```
Since every set $U \subset \mathbb{R}$ is contained in an interval of the same diameter, it follows that
```math
\mu(U)\le a^{-s}(\operatorname{diam}(U))^s
\qquad
\text{whenever } 0 \lt \operatorname{diam}(U) \lt a.
```
Lemma 2.1 now gives $\dim_H(C_a)\ge s$.

Thus $\dim_H(C_a)=s$.

**Upper box dimension.**

Let $0 \lt \delta \lt 1$, and choose $n \in \mathbb{N}$ such that
```math
r^{n+1} \lt \delta \le r^n.
```
Then the $2^{n+1}$ level-$(n+1)$ intervals have diameter $r^{n+1} \lt \delta$, so they form an admissible cover. Hence
```math
N_\delta(C_a)\le 2^{n+1}.
```
Therefore
```math
\frac{\log N_\delta(C_a)}{-\log\delta}
\le
\frac{(n+1)\log 2}{-n\log r}.
```
Letting $\delta\downarrow 0$ (equivalently $n\to\infty$) yields
```math
\overline{\dim}_B(C_a)\le s.
```

**Lower box dimension.**

Let $0 \lt \delta \lt a/2$, and choose $n \in \mathbb{N}$ such that
```math
\frac a2\,r^n \lt \delta \le \frac a2\,r^{n-1}.
```
Any set of diameter at most $\delta$ has diameter strictly smaller than the smallest gap $ar^{n-1}$ between level-$n$ intervals. So such a set can meet at most one level-$n$ interval. Since there are $2^n$ level-$n$ intervals, every admissible $\delta$-cover must use at least $2^n$ sets:
```math
N_\delta(C_a)\ge 2^n.
```
Hence
```math
\frac{\log N_\delta(C_a)}{-\log\delta}
\ge
\frac{n\log 2}{-\log(a/2)-n\log r}.
```
Letting $\delta\downarrow 0$ gives
```math
\underline{\dim}_B(C_a)\ge s.
```

Since always $\underline{\dim}_B\le \overline{\dim}_B$, we conclude that
```math
\underline{\dim}_B(C_a)=\overline{\dim}_B(C_a)=s.
```
This completes the proof.

∎

**Lemma 2.3.**

For every bounded set $E \subset \mathbb{R}$ and every $\delta \gt 0$,
```math
N_\delta(E)=N_\delta(\overline{E}).
```
Consequently,
```math
\underline{\dim}_B(E)=\underline{\dim}_B(\overline{E}),
\qquad
\overline{\dim}_B(E)=\overline{\dim}_B(\overline{E}).
```

**Proof.**

If $\overline{E}\subset\bigcup_{j=1}^m U_j$, then certainly $E\subset\bigcup_{j=1}^m U_j$, so
```math
N_\delta(E)\le N_\delta(\overline E).
```
Conversely, if $E\subset\bigcup_{j=1}^m U_j$ and each $U_j$ has diameter at most $\delta$, then
```math
\overline E\subset\bigcup_{j=1}^m \overline{U_j},
```
and $\operatorname{diam}(\overline{U_j})=\operatorname{diam}(U_j)\le \delta$. Hence
```math
N_\delta(\overline E)\le N_\delta(E).
```
So the covering numbers are equal, and the box dimensions are therefore unchanged by taking closure.

∎

**Corollary 2.4.**

Let $\alpha \in (0,1)$ and $\rho = (1-\alpha)/2$. Then
```math
\begin{aligned}
\dim_H(C_\alpha^+) &= \dim_H(C_\alpha^-) \\
&= \underline{\dim}_B(C_\alpha^+) = \overline{\dim}_B(C_\alpha^+) \\
&= \underline{\dim}_B(C_\alpha^-) = \overline{\dim}_B(C_\alpha^-) \\
&= \frac{\log 2}{\log(2/(1-\alpha))}.
\end{aligned}
```

**Proof.**

By Theorems 1.6 and 1.7,
```math
C_\alpha^+ = C_\alpha\setminus E_\rho^+,
\qquad
C_\alpha^- = C_\alpha\setminus E_\rho^-.
```
The sets $E_\rho^+$ and $E_\rho^-$ are countable, because they are indexed by finite binary words.

**Hausdorff dimension.**

We prove the claim for $C_\alpha^+$; the proof for $C_\alpha^-$ is identical. Since $C_\alpha^+\subseteq C_\alpha$, we have
```math
\dim_H(C_\alpha^+)\le \dim_H(C_\alpha).
```
For the reverse inequality, let $t \gt \dim_H(C_\alpha^+)$. Then $\mathcal{H}^t(C_\alpha^+)=0$. Also $\mathcal{H}^t(E_\rho^+)=0$ because $E_\rho^+$ is countable. Hence
```math
\mathcal{H}^t(C_\alpha)
\le
\mathcal{H}^t(C_\alpha^+) + \mathcal{H}^t(E_\rho^+)
=
0,
```
so $t \ge \dim_H(C_\alpha)$. Since this holds for every $t \gt \dim_H(C_\alpha^+)$, we obtain
```math
\dim_H(C_\alpha)\le \dim_H(C_\alpha^+).
```
Therefore $\dim_H(C_\alpha^+) = \dim_H(C_\alpha)$. The same argument gives $\dim_H(C_\alpha^-) = \dim_H(C_\alpha)$.

**Box dimension.**

Again we treat $C_\alpha^+$; the proof for $C_\alpha^-$ is the same. Since $\pi_\rho:\Sigma\to C_\alpha$ is a homeomorphism and $E_\rho^+$ is countable, its preimage
```math
S^+ := \pi_\rho^{-1}(E_\rho^+)
```
is countable. Every nonempty cylinder in $\Sigma$ is uncountable, so $\Sigma\setminus S^+$ is dense in $\Sigma$. Therefore
```math
C_\alpha^+
=
C_\alpha\setminus E_\rho^+
=
\pi_\rho(\Sigma\setminus S^+)
```
is dense in $C_\alpha = \pi_\rho(\Sigma)$, and hence
```math
\overline{C_\alpha^+} = C_\alpha.
```
Lemma 2.3 now yields
```math
\underline{\dim}_B(C_\alpha^+) = \underline{\dim}_B(C_\alpha),
\qquad
\overline{\dim}_B(C_\alpha^+) = \overline{\dim}_B(C_\alpha).
```
The same holds with $C_\alpha^-$ in place of $C_\alpha^+$.

Finally, Theorem 2.2 applied to $a = \alpha$ gives
```math
\dim_H(C_\alpha)
=
\underline{\dim}_B(C_\alpha)
=
\overline{\dim}_B(C_\alpha)
=
\frac{\log 2}{\log(2/(1-\alpha))}.
```
Substituting this into the equalities above completes the proof.

∎
